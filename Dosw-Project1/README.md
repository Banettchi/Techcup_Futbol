# Dosw-Project1

## Systems Soccer Tournament

Proyecto de desarrollo de software para el torneo de fútbol.

## Estructura del Proyecto

```
Dosw-Project1/
├── pom.xml
└── src
    ├── main
    │   └── java
    │       └── com/example/App.java
    └── test
        └── java
            └── com/example/AppTest.java
└── docs
    ├── uml/
    ├── images/
    ├── requirements/
    ├── planning/
```

## Requisitos

- Java 21
- Apache Maven 3.9+

## Ejecución

```bash
mvn compile
mvn test
```
